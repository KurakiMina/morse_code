# モールス信号練習

提示された旧UIコードと参考画像をベースに、クラシックモードをボード型UIとして復元した版です。

## ファイル

- `index.html`
- `styles.css`
- `app.js`

## 実装内容

- モード選択
- クラシックモード
  - 参考画像に寄せた `MORSE CODE` ボード
  - ● = 短押し
  - ■ = 長押し
  - 通過済み・現在位置を塗りつぶし
  - 数字チップを下段に表示
- ゲームモード
  - 40秒制限
  - ランダム単語
  - 1文字ずつ判定
  - スコア、コンボ、ランキング5位保存

## GitHub Pages

リポジトリ直下に3ファイルを配置してください。

```text
index.html
styles.css
app.js
```


## 今回の復旧

- クラシックモードは固定
- 練習モードを復活
- ゲームモードUIを復旧
- ゲームモードに40秒タイマー、スコア、コンボ、結果画面、ランキングを再追加


## クラシックモード調整

- O/M/Q/G/Z 周辺を修正
- Q の重複表示を解消
- Z を正しいモールスコード `--..` 側に配置
- `----` 側の誤ったQ表示を非表示化
